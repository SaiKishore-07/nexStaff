# 🚀 NexStaff - Employee Management System

NexStaff is a modern and responsive **Employee Management System** built using **React.js**, **Tailwind CSS**, **React Router DOM**, and **Context API**. It allows users to efficiently manage employee records with features like adding, editing, deleting, searching, and filtering employees. Employee data is stored in **Local Storage**, ensuring persistence even after refreshing the browser.

## ✨ Features

- 📊 Dashboard with employee statistics
- 👥 View all employees
- ➕ Add new employee
- ✏️ Edit employee details
- 🗑️ Delete employee with confirmation modal
- 🔍 Search employees by Name or Employee ID
- 🏢 Filter employees by department
- 💾 Local Storage support
- 📱 Fully Responsive Design
- ⚠️ Empty State UI
- 🚫 Custom 404 Not Found Page

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/nexstaff.git
```

### 2️⃣ Navigate into the project

```bash
cd nexstaff
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start the development server

```bash
npm run dev
```
